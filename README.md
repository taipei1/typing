# Тренажер печати

Минималистичное одностраничное приложение для тренировки набора английских слов по русскому переводу. Данные хранятся локально в браузере.

## Возможности

- **Сессии по таймеру** — 1, 2 или 5 минут; после старта показываются переводы, нужно напечатать английский вариант.
- **Озвучивание** — опционально через Web Speech API (громкость настраивается).
- **Фильтр слов для тренировки** — кнопка «Фильтр»: по весу карточки, диапазону дат добавления, последним N добавленным словам, минимальной длине слова (букв).
- **Словарь** — массовое добавление строк вида `english[TAB]перевод`, таблица с редактированием, экспорт в JSON.
- **Повторение** — у карточек есть вес и статистика (упрощённая SRS-подобная логика в коде).

## Запуск

Откройте `index.html` в браузере или поднимите локальный сервер:

```bash
python -m http.server 8765
```

Либо запустите `run-server.bat` в каталоге проекта и откройте в браузере `http://127.0.0.1:8765/`.

## Структура

| Файл | Назначение |
|------|------------|
| `index.html` | Разметка и стили страницы |
| `app.js` | Логика тренажёра, хранилище, фильтры |
| `run-server.bat` | Запуск простого HTTP-сервера на Windows |

## Данные

Словарь и рекорды сохраняются в `localStorage` ключами вида `typing-trainer-vocab-v2` и `typing-trainer-best-score-v3`. Резервную копию можно сделать через «Скачать JSON» в интерфейсе.

## Требования

Современный браузер с поддержкой JavaScript и localStorage. Для озвучивания — встроенный синтез речи (зависит от ОС и браузера).
